JobFiller Help & Instructions

Learn how to use JobFiller to streamline your job applications
Getting Started

Welcome to JobFiller!

JobFiller is a Chrome extension that makes applying for jobs easier by automatically filling out application forms using your resume data. This guide will help you understand how to set up and use JobFiller effectively.

1
Upload Your Resume

Before you can start using JobFiller, you need to upload your resume:

  1. Click the JobFiller icon in your Chrome toolbar and select Manage Resumes
  2. In the options page, click the Upload Resume tab
  3. Enter a name for your resume (e.g., "Software Developer Resume 2025")
  4. Drag and drop your resume PDF file or click to browse
  5. Click Upload Resume

JobFiller will parse your resume and extract all the important information, including your personal details, work experience, education, skills, and more.

2
Fill Out Job Applications

When you're on a job application page:

  1. Click the JobFiller icon in your Chrome toolbar
  2. Select the resume you want to use from the dropdown (if you have multiple)
  3. Click Autofill This Application

JobFiller will automatically fill out the form fields with information from your resume. You can then review and make any necessary adjustments before submitting.

3
Map Fields for Custom Sites

If JobFiller doesn't correctly identify form fields on a particular job site:

  1. Click the JobFiller icon in your Chrome toolbar
  2. Click Map Fields for This Site
  3. For each form field, select the corresponding resume field from the dropdown
  4. Click Save Mappings

These mappings will be saved for the specific website, so you won't need to map fields again for future applications on the same site.

Pro Tip

You can upload multiple resumes and switch between them for different types of job applications. For example, you might have one resume for technical roles and another for management positions.

Advanced Features

Multiple Resume Management

JobFiller allows you to store and manage multiple resumes:

  • Upload different versions tailored to specific job types or industries
  • Set a default resume that will be used automatically
  • View and edit your resume data directly in JSON format
  • Delete outdated resumes when they're no longer needed

Custom Field Mappings

Different job sites use different field names and structures. JobFiller allows you to create custom mappings for each site:

  • Map resume fields to form fields based on their labels or IDs
  • Save mappings per domain, so they apply automatically on future visits
  • Review and update mappings as needed

Auto-fill on Page Load

You can configure JobFiller to automatically fill out forms as soon as you open a job application page:

  1. Click the JobFiller icon and select Manage Resumes
  2. Go to the Settings tab
  3. Check the Automatically fill forms when page loads option
  4. Click Save Settings
Important Note

Auto-fill on page load is convenient but may not work perfectly on all sites. It's always a good idea to review the filled information before submitting an application.

Troubleshooting

Form Fields Not Being Filled

If JobFiller isn't filling certain fields correctly:

  1. Check if the field is visible and enabled on the page
  2. Use the "Map Fields for This Site" feature to create a custom mapping
  3. Make sure your resume contains the required information
  4. Some fields may require manual filling (like file uploads or captchas)

Resume Parsing Issues

If your resume isn't being parsed correctly:

  1. Make sure your PDF is text-based and not a scanned image
  2. Try using a simpler resume format with clear section headers
  3. You can manually edit the parsed resume JSON in the Options page
  4. Alternatively, create a JSON resume directly using the editor in the Options page

Extension Not Working

If JobFiller isn't working at all:

  1. Make sure the extension is enabled in Chrome
  2. Check that you have at least one resume uploaded
  3. Try refreshing the page before using the extension
  4. Some sites use complex forms that may not be fully compatible
Privacy & Security

JobFiller is designed with your privacy in mind:

  • All your resume data is stored locally in your browser storage
  • No data is sent to any external servers or third parties
  • The extension only operates on job application websites
  • Resume parsing happens entirely within your browser

To ensure maximum security:

  • Keep your Chrome browser and the JobFiller extension updated
  • Review the permissions requested by the extension
  • Always check the filled information before submitting applications
